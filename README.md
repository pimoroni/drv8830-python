# Installing

Stable library from PyPi:

* Just run `python3 -m pip install drv8830`

Latest/development library from GitHub:

* `git clone https://github.com/pimoroni/drv8830-python`
* `cd drv8830-python`
* `sudo ./install.sh --unstable`
